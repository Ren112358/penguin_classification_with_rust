## Overview
Make a model to solve Palmer penguin classification with Rust.

## Installation
```bash
git clone git@github.com:Ren112358/penguin_classification_with_rust.git
```

## Usage

```bash
❯ cd src

❯ cargo run
accuracy : 0.9509803921568627
```
